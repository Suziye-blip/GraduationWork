## サービス概要
架空の人物を答えさせるアキネイタークイズアプリを作りたい。

## このサービスへの思い・作りたい理由
アキネイタークイズ企画をyoutubeのショート動画で見ることが好きなので、自分もやってみたいと思った。  
しかし、性質上参加者が最低2人以上いなければ実施することができない。  
また、実在する人物や事物を問題とする場合、その実際のデータを集めるとなると膨大な作業になるし、正誤判定するアルゴリズムもかなり複雑なものになることが予想される。  
一方で、「Return of the Obra Dinn」に代表されるような「人物当てゲーム」も作りたいと考えた。  
そこで、この2つの要素を混ぜてしまえばよいのではないかという結論に至った。  
すなわち、Fakerなどによって年齢、性別、出身地、趣味などを設定した複数の架空の人物のダミーデータを出力させて、「その人の〇〇は××ですか？」の形式で回答してもらう形式にすればよいのではないかと考えた。  

### アキネイタークイズとは
アキネイタークイズは、クイズの一種。  
ルールは以下の通り。  
1. 出題者は人物、物、地名といった特定の事物を回答として1つ設定する。
2. 回答者はそれについての質問を出題者に1つずつ行う。
3. 出題者はその質問に対して「はい」「いいえ」「どちらでもない」などで答える。
4. その出題者の回答をヒントに、出題者の定めた回答を答える。
この形式のクイズは古くからある（1947-1959にNHKラジオにて放送された「二十の扉」など）が、近年ではYouTube上で同じルールのクイズが「アキネイタークイズ」として、企画動画が多く投稿されている。  

### Return Of The Obra Dinnとは  
Return Of The Obra Dinnは、ルーカス・ポープ製作のアドベンチャーゲーム。  
1807年、5年前に消息を絶った英国の商船、オブラディン号を舞台としている。  
保険調査官である主人公は船内の状況や手掛かりから、当時の乗員乗客の身元や安否を調べることになる。  

## ユーザー層について
少し頭をゲームを使ったゲームをしてみたいユーザー、もしくは3、4人程度でワイワイ楽しみたい小集団（世代や性別によるターゲット層の違いは特に考慮していない）。

## サービスの利用イメージ
- 1回数分程度で遊ぶことができて、人物当てゲームを楽しむことができる。
- 慣れてきたらより少ない質問で当ててみようというモチベーションが出てくる。

## サービスの差別化ポイント・推しポイント
競合アプリA（みんなで推理）  
メリット：AIが出題するため、一人でもプレイをすることができる。  
デメリット：ハルシネーションが発生し、正しい答えにたどり着けない場合がある（それにより一層盛り上がる場合もある）。  

競合アプリB（Return Of The Obra DinnやThe Case of the Golden Idol、Type Helpといったフォロワー作品）  
メリット：一人でじっくり腰を据えてプレイすることができる。重厚な世界観や設定。  
デメリット：一度クリアしてしまったら謎解き部分はそれで終わり。複数人でプレイすることができない。  

自分のアプリの差別化ポイント  
- 問題はGemのダミーデータによって生成される→何度でもプレイが可能かつ実装も容易。
- 1プレイは数分で終わるので、サクッとプレイすることができる（短い手で正答するのであれば腰を据えて考えてもいい）。  
- ターン制による複数人プレイをすることもできる→答えに迫りすぎる質問をしてしまうと相手にヒントを与えてしまうという駆け引きが発生する。  

## 機能候補
- MVPリリース時：複数人の架空の人物のプロフィールを作り、それについて「その人の〇〇は××ですか？」と質問して絞りこんで最終的に回答できるまでの一連の仕組み。  
- 本リリース時：
    - ターン制にして複数人で遊べる仕組み
    - 少ない手数で正答したときほど高得点になる仕組み
    - 難易度設定
        - レベル制（レベルが低ければ、候補者や情報の数が少なくなる。レベルが高ければ、数が多くなる）。
            - 問題を解けば解くほど、どんどんレベルが上がっていく仕組み。
        - 情報を隠すオプションを付ける。
        - 候補者の数を任意で設定できる。
        - 一度に複数人を答えさせる(このタイプのアキネイタークイズについては'https://www.youtube.com/shorts/QVvg-_-frxU'を参照)。
    - SNSで結果をシェアできる仕組み
